About me

I'm currently a fire security agent in a large company, and I've learnt how to prioritise what needs to be done. For the convenience of the team I'm part of, I wanted to develop an application to simplify a few tasks for us. That's where my passion for programming began!

Since then, in my spare time, I've been training in languages that, I think, will have a future and that I like, such as JavaScript, Angular and Typescript. Having had the opportunity to create my school's website in 1996, I've also relearned HTML and learned CSS.

I'm lucky enough to be able to take part in an Open Source project to create an Udemy-type web application, but with additional specifications, and we hope that the project will come to an end and work! The next step for the project is for me to train in .net so that I can also work on the back-end. Whenever I have a moment available, I either start learning or developing, because I find it very fulfilling. Since mid-June 2024, I've been focusing my free time on learning through a certification course at Skill&You.
